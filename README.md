# Music Theory Engine

Interactive music theory learning tool for Windows.

## Features

- Piano roll visualization
- Fretboard visualization
- Circle of fifths
- Stave/notation display
- MIDI engine for audio playback
- Interactive widgets for hands-on learning

## Download

Grab the latest installer from the [Releases](https://github.com/acquah0/Music-Theory-Engine/releases) page.

## Requirements

- Windows 10 or later
- 64-bit system

## Installation

Run `MusicTheoryEngine_Setup.exe` and follow the installer prompts. No additional dependencies needed.
